# homebrew-tap

Homebrew formulae for [xbglowx](https://github.com/xbglowx) projects.

## Usage

```sh
brew tap xbglowx/tap
```

Or install directly without tapping first:

```sh
brew install xbglowx/tap/<formula>
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [vault-kv-search](https://github.com/xbglowx/vault-kv-search) | Recursively search secrets in HashiCorp Vault KV stores |
| [vault-kv-mv](https://github.com/xbglowx/vault-kv-mv) | Move and rename secrets in HashiCorp Vault KV stores |
| [github-org-repos-sync](https://github.com/xbglowx/github-org-repos-sync) | Sync all repositories from a GitHub organization locally |
